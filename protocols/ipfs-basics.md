# 8% Summary: The Core of IPFS

## 1. What is IPFS?
The InterPlanetary File System (IPFS) is a decentralized, peer-to-peer file-sharing network. Instead of downloading a file from a single location (like an AWS cloud server), you download pieces of it from multiple nodes around the world simultaneously.

## 2. The Core Mechanic: Content Addressing
* **Web2 (Location-Based):** You look up a file by *where* it is hosted (e.g., `https://server.com/photo.jpg`). If the server goes down, the file vanishes.
* **Web3 (Content-Based):** You look up a file by *what* it is. Every file uploaded to IPFS gets a unique cryptographic fingerprint called a **CID (Content Identifier)**. If the file content changes by even one pixel, the CID changes completely.

## 3. Why It Matters for Vibe Coders
* **Censorship Resistant:** Once a file or static website is pinned to IPFS, it cannot be taken down or altered by a single central entity.
* **Permanent Portfolios:** Hosting web apps via IPFS means your projects live on a global distributed network forever, completely free of traditional hosting subscription fees.
